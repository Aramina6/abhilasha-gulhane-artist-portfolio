# Abhilasha Gulhane — Artist Portfolio

Static HTML portfolio site for Abhilasha Gulhane, deployed to GitHub Pages.

This version was built directly from the original `Abhilasha_Gulhane_Artist_Portfolio_Optimized.pdf` (created with reportlab).

## What's included
- Clean, modern single-page professional site (`index.html`)
- The 4 exact cleaned artworks with white borders, extracted from the PDF:
  - The Loom of Life
  - The Collective Current
  - Trilogy of Consciousness (triptych)
  - The Seer (Devi of Patterns)
- Full accurate "About the Artist" bio
- Full accurate Artist Statement
- Responsive gallery with elegant white-frame presentation and full-screen lightbox showing titles + detailed descriptions
- Original PDF included for download
- Fully static (no build tools required)

## Live site
https://Aramina6.github.io/abhilasha-gulhane-artist-portfolio/

## How to update in the future
1. Edit `index.html` (or replace images in `images/`)
2. Run the usual git add / commit / push
3. GitHub Pages will auto-deploy

## Local preview
Just open `index.html` in any browser.
